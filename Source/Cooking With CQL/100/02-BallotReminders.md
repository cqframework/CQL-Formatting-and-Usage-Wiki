# Ballots Are Open

Reminder that the ballots for the 2026 May Cycle are open, and will be open through May 11th.

The following CQL-relevant ballots are in progress:

## Quality Measure IG UV STU2

[Quality Measure IG UV STU2 Ballot](https://hl7.org/fhir/uv/cqm/2026May/en/)

Key Changes

* Clarified support for [value-based stratifiers](https://hl7.org/fhir/uv/cqm/2026May/en/measure-conformance.html#stratification)
* Expanded support for [reporting timing](https://hl7.org/fhir/uv/cqm/2026May/en/StructureDefinition-cqm-datatype-timing.html)
* Subscription support for [measure changes](https://hl7.org/fhir/uv/cqm/2026May/en/Bundle-EmergencyCapacityReportingExample.html)
* Added [Supporting Evidence](https://hl7.org/fhir/uv/cqm/2026May/en/Measure-SupportingEvidenceExample.html)
* Risk-adjusted Measure Example [CAUTI SIR](https://hl7.org/fhir/uv/cqm/2026May/en/Measure-CAUTISIRExample.html)

## Data Exchange for Quality Measures UV STU1

[Data Exchange for Quality Measures UV STU1](https://hl7.org/fhir/uv/deqm/2026May/en/)

Key Changes

* Clarified exchange and reporting use cases
* Submit Data as a Bundle POST
* Aligned Submit Data with Upcoming Bulk IG
* Support for [Invited Pull](https://hl7.org/fhir/uv/deqm/2026May/en/OperationDefinition-collect-data.html)
* Generalized to Universal Realm
